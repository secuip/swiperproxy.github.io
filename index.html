SWIPERPROXY

A lightning-fast, open source web proxy that is easy for you to run and customize.

Download » Live Demo »

 Lightning-fast

SwiperProxy runs on efficient Python code to run a self-containing, minimalist webserver. Prefer your own? SwiperProxy plays well and scales with major HTTPD's including Apache, Nginx and Varnish

 Easy to set up

Only 25 lines of well-documented configuration are needed to run SwiperProxy, yet is fully featured supporting HTTP, HTTPS, blocklists, URL rewrites, logging and more.

 Even easier to use

If you don't want to deal with TCP-ports, SOCKS-proxies or Tor bridges then SwiperProxy is perfect for you. Simply enter the URL you want to visit, or browse directly to your proxied destination.
