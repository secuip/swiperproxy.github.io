Stay informed of any security issues we find

We take security very seriously, but we also know no software is perfect. Inevitably, security issues will be discovered. Of course, we will release a patch as soon as possible. We believe in responsible full disclosure, and you can sign up to receive an e-mail whenever a security issue is discovered.
